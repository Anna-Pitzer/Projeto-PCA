  <h1>Sistema CRUD de alunos em Python</h1>
</div>

<h2>Sobre o projeto:</h2>
<p>É um sistema de gerenciamento de alunos desenvolvido como parte da avaliação prática (Av2) da disciplina de Princípios de construção de algoritmos, com o objetivo de aplicar e demonstrar domínio sobre a manipulação de dados e boas práticas de programação.</p>
<h2>Objetivos específicos:</h2>
<p><ul>
  <li>Aplicar os conceitos aprendidos em sala da biblioteca Pandas</li>
  <li>Cronstruir um sistema CRUD funcional e modular</li>
  <li>Organizar dados de forma correta</li>
</ul></p>
<h2>Funcionalidades:</h2>
<p><ul>
  <li>Cadastro de alunos em arquivo .csv</li>
  <li>Pesquisa por nome ou matrícula</li>
  <li>Editação de registros existentes</li>
  <li>Remoção com confirmação</li>
</ul></p>
<h2>Tecnologias Utilizadas:</h2>
<p><ul>
  <li>Python</li>
  <li>Pandas</li>
  <li>Git (Controle de versão)</li>
</ul></p>
<h3>Como executar:</h3>
<p>Baixe o repositório e extraia os arquivos. No terminal, navegue até a pasta  <code>sistema_de_alunos</code> e execute: pip install pandas, espere que o download seja terminado e execute: python __main__.py após isso o sistema iniciará.</p>
